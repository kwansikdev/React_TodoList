# React_TodoList

<br />

---

## 기능

1. Todo 추가 
2. Todo 삭제
3. Todo 상태 체크(Completed)
4. 완료된 Todos 한번에 삭제
5. 전체 Todos 상태 한번에 체크 
6. 완료상태에 따른 Todos 카운팅
7. Tab(All / Active / Completd)에 따른 Todo 분류
